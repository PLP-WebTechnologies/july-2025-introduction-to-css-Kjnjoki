About This Page

This page demonstrates how the CSS Box Model works with margins, borders, padding, and content. It also demonstrates selectors and consistent styling.

Box Model Elements

  • Content – The text, image, or data inside the element.
  • Padding – Space between content and border.
  • Border – A line surrounding the padding.
  • Margin – Space between elements.
